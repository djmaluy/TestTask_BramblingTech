### Installation:

- **To clone the project**

```
git clone git@github.com:djmaluy/TestTask_BramblingTech
```

- **Go to the folder**

```
cd TestTask_BramblingTech.git
```

- **Execute command**

```
npm install
```

### Run the application:

- **For quick application run use this command**

```
npm start
```
